☁️ Configurar Google Drive

Per exportar a Drive necessites un Client ID de Google i l'ID de la carpeta on vols desar els fitxers.

🔗 Obtenir Client ID a Google Cloud Console
  1. Crea un projecte nou
  2. Activa l'API de Google Drive
  3. Crea credencials → OAuth 2.0 → Aplicació web
  4. Afegeix l'URL d'aquesta app als orígens autoritzats
  5. Copia el Client ID aquí
Per l'ID de carpeta: obre la carpeta a Drive, copia el codi de la URL: drive.google.com/drive/folders/AQUEST_ID
